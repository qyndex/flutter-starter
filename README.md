# Flutter Starter

Material 3 Flutter starter app with bottom navigation, GoRouter routing, and Riverpod state management across 3 screens.

**Stack:** Flutter | **Complexity:** simple
**Tags:** `flutter`, `mobile`, `material3`, `gorouter`, `riverpod`, `starter`, `dart`

## Quick Start

```bash
flutter pub get
flutter run
```

## Claude Code

This project includes a `CLAUDE.md` file optimized for development with
